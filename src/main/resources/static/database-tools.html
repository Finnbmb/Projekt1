Verwenden Sie die Database View für diese häufigen Abfragen:

  • Alle Benutzer: SELECT * FROM users
  • Termine eines Benutzers: SELECT * FROM appointments WHERE user_id = ?
  • Heutige Termine: SELECT * FROM appointments WHERE DATE(start_date_time) = CURRENT_DATE
  • Feiertage: SELECT * FROM holidays WHERE date >= CURRENT_DATE
